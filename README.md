# BRNV Currency Converter

A polished static currency converter website for GitHub Pages. Built with plain HTML, CSS, and vanilla JavaScript.

## Features

- Live two-way currency conversion
- EUR to USD as the default pair
- Currency selector with search, flags, codes, and full names
- English by default with a Ukrainian language toggle
- Light and dark theme support
- Popular conversion cards with compact rates
- Google Analytics 4 placeholder
- Google AdSense-ready banner placeholder
- SEO meta tags, Open Graph/Twitter tags, sitemap, robots.txt, FAQ content, and JSON-LD structured data

## Hosting

This project is fully static and can be hosted on GitHub Pages. No build step, backend, Node.js, React, Vue, or Next.js is required.

Open `index.html` locally or publish the repository with GitHub Pages.

## Configuration

Replace these placeholders before production:

- `G-BPPLNFQSNS` with a different GA4 Measurement ID if you change analytics properties
- `ca-pub-XXXXXXXXXXXXXXXX` with your real AdSense publisher ID
- `data-ad-slot="XXXXXXXXXX"` with your real AdSense slot ID
- `https://barik421.github.io/BRNV-Currency-Converter/` with your custom domain in canonical URLs, Open Graph URLs, JSON-LD, `robots.txt`, and `sitemap.xml`
