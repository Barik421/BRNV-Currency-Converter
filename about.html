About BRNV Currency Converter

A simple tool with a clean product focus

BRNV Currency Converter is a lightweight static website for fast informational currency conversion. It is designed to feel clean, premium, and easy to use on both mobile and desktop.

The project was created by Maksym Baranovych as part of a growing collection of practical web utilities and Chrome extension projects under the BRNV direction.

The idea is simple: build small, polished tools that do one job well, without unnecessary complexity, accounts, or backend infrastructure.

More projects

You can find more projects and experiments on GitHub. The repository profile includes other work related to web tools, browser extensions, and static product-style interfaces.
